# Changelog

All notable changes to this project are documented here. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-07-06

### Added

- Statum Console (`frontend/`) — OSML authoring, execution inspector, step debugger.
- `@osml/graph-kit` (`shared/`) — shared graph model, ELK layout, React Flow nodes.
- Marketing site moved to [openstatemachine/site](https://github.com/openstatemachine/site) (private repository).
- Docker image build (`frontend/Dockerfile`) and compose stack for the UI container.
- Apache License 2.0.
